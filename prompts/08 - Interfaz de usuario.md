Rol
Eres un diseñador senior de UX/UI especializado en aplicaciones nativas para macOS, con dominio de las Human Interface Guidelines de Apple.

Tarea
Diseña la interfaz de un emulador de Super Nintendo (SNES) para macOS. Prioriza la simplicidad, la velocidad de uso y una estética minimalista sin sacrificar funcionalidades para jugadores frecuentes.

Referencia
Analiza el wireframe adjunto y úsalo como base de la composición.

Alcance
Una sola vista del emulador basada en el wireframe. Durante el juego, mostrar únicamente el contenido del juego.

Principios de diseño
- Lenguaje visual nativo de macOS
- Jerarquía visual clara
- Uso generoso del espacio en blanco
- Sin elementos decorativos innecesarios

Especificaciones visuales
- Tipografía San Francisco
- Iconografía SF Symbols
- Colores neutros
- Barras laterales nativas
- Transparencias y materiales de macOS
- Animaciones sutiles
- Escalado HiDPI
- Relación de aspecto original de SNES
- Compatible con modo claro y oscuro, con tema oscuro por defecto

Calidad
Acabado profesional, comparable con una aplicación comercial distribuida en macOS.

# Tarea
Quiero usar snes9x_libretro como núcleo de emulación en mi aplicación. Indica dónde descargar los binarios oficiales más recientes para Windows (.dll) y macOS (.dylib).

Para cada plataforma proporciona la URL oficial de descarga y el nombre del archivo del núcleo.

# Verificación obligatoria
Las URL y los nombres de archivo cambian con el tiempo y un enlace incorrecto es un riesgo de seguridad. Por tanto:
- Verifica la información en la web antes de responder. Si no tienes acceso a la web, dilo de forma explícita y no inventes URLs ni nombres de archivo.
- Acepta como oficial únicamente la distribución del proyecto Libretro, por ejemplo su buildbot o su organización en GitHub, o la del proyecto Snes9x. Descarta sitios de terceros.
- Para cada dato, indica de qué fuente oficial procede.
- Si la distribución oficial no ofrece un binario directo para alguna plataforma y requiere otra vía de obtención, indícalo en lugar de forzar una URL.
